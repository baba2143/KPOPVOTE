# K-VOTE COLLECTOR プロジェクト概要

## プロジェクト目的

**K-VOTE COLLECTOR** は、K-POPファン向けの外部投票情報一元管理アプリケーションです。

### 核となる価値
- **外部投票情報の一元管理**: 複数の投票アプリ・サイトの情報を1箇所で管理
- **推し別タスク効率化**: メンバー単位で推し設定し、関連投票を自動フィルタリング
- **投票抜け漏れ防止**: 締め切り通知と進捗管理で投票機会を逃さない

### ターゲットユーザー
- 10代〜30代の熱心なK-POPファン
- 複数の投票サイト・アプリを使用しているユーザー
- 推しメンバーの投票を効率的に管理したいファン

### 開発フェーズ
- **現在**: Phase 0（企画・設計段階）
- **次**: Phase 1（iOS MVP開発 - 3ヶ月）
- **将来**: Phase 2（Android MVP開発 - 2ヶ月）

## MVP コア機能

### 1. 投票タスク管理
- 外部URL登録（IDOL CHAMP, Mnet Plus, MUBEAT等）
- OGP画像自動取得
- 締め切り管理とソート
- 進捗ステータス管理（未投票/投票済み/ポイント不足）
- プッシュ通知

### 2. パーソナライズ機能
- メンバー単位の推し設定（myBias）
- 推しベースのホーム画面フィルタリング
- 推し専用タスクリスト

### 3. コミュニティ機能
- 推しメンバー別コミュニティ
- 投稿・いいね・コメント機能
- ファン同士の情報共有

### 4. アプリ内独自投票
- オリジナル投票の実施
- 無料ポイント制での投票
- リアルタイム集計結果表示
